# Welcome to browsemark

Point browsemark at any directory of markdown files — your project docs, specs, ADRs, runbooks — and browse them in a three-panel IDE-like layout with a file tree, rendered viewer, and document outline.

**Click any element** (heading, paragraph, code block, list item) to annotate it. Your annotations flow directly to AI coding agents like Claude Code, Cursor, and VS Code Copilot via MCP — no copy-paste needed. Review a doc, leave feedback, and your AI agent picks it up automatically.

## Rendering Features

Browse the file tree on the left to see these in action:

- **[[wiki-links]]** — `[[wiki-links]]` (Obsidian-compatible) that resolve and navigate between files
- **[[mermaid-diagrams]]** — Flowcharts, sequence diagrams, and more with live re-rendering
- **[[math-and-code]]** — KaTeX math formulas and syntax-highlighted code blocks
- **[[tables-and-alerts]]** — GFM tables and GitHub-style alert callouts
